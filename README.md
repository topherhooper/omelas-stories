# Omelas Stories Podcast

A D&D actual play podcast.

## Subscribe

Add this RSS feed URL to your favorite podcast app:

```
https://topherhooper.github.io/omelas-stories/feed.xml
```

Works with:
- Apple Podcasts
- Pocket Casts
- Overcast
- Spotify (via RSS)
- Any podcast app that supports RSS feeds

## Setup

1. Replace `topherhooper` in `feed.xml` with your actual GitHub username
2. Replace `YOUR_EMAIL@example.com` with your email (required by some podcast directories)
3. Add `artwork.jpg` (1400x1400 to 3000x3000 pixels recommended)
4. Enable GitHub Pages in repository settings (serve from main branch)
